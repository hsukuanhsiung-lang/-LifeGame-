# -LifeGame-
  改动历史详情

  v1
改动目的：优化表世界操作界面，完善功能。

  Bug：
1无法删除“时间”内小项。

  调整：
1缩小挑战內项目大小，让用户不用下拉屏幕的情况下显示8个每日任务和前2个禁止事项。

  新添功能：
1正计时和倒计时都有“暂停”和“结束”两个选项。
2正计时倒计时同项目累计最低时间单位15分钟以后自动转换获得0.25属性。
3正计时和倒计时期间玩家被锁定在此界面，在用户点击正计时和倒计时以后就不能切换界面了，除非点击“结束”或者“暂停”。
4用户点击禁止事项以后询问用户确定，例如玩家第一次点戒烟，询问玩家是否确定戒烟询问玩家戒烟开始日期。
5“时间”里显示现在时间那个加亮的格子在保持原本功能的情况下在当前时间格子显示当前时间，格式为00:00。
6“记忆”内像“时间”的大项分类一样给不同来源的信息做分类，“时间”为一类，“行动”为一类，“挑战”为一类，“任务”为一类。有小项的增加小项选项可以点击分类查看。“记忆”内可以看到大分类“时间”“行动”“挑战”“任务”，例点击大项挑战以后，弹出小项“每日任务”“禁止事项”，布局方式沿用原有“记忆”布局方式。



  v1.1
  改动目的：优化完善。
  
  Bug：
1“时间”里面标记的时间块的小项，对应到记忆里面具体的时间数字是错的，无论是多长时间都只显示为0.25小时

  新添功能：
1“时间内”选了大项，小项，然后点击格子后，增加小项的内容在格子上。例点击健身以后再点击21:00-22:00的格子，那这四个格子便在原有变成红色的基础上加上健身两个字。
2任务的子任务也会获得奖励，也有等级区分。
3子任务默认折叠，点击展开以后才会保持展开。
4里世界下面增加一个冒险者公会，冒险者公会定期高价值发布悬赏任务。
5玩家根据完成的任务数量积累声望，用以提高冒险者登记。冒险者等级如任务难度等级一样分八种登记。
6E级冒险者练习三天不登录会被降级为F级。
